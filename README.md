# JavaEE7 and Payara Micro RESTFul example with Basic Authentication and Arquillian Test Suite

## Index

1. **Setup**
2. **Running instructions**
3. **API Documentation**

## 1.Setup

* Java 7+ JDK
* Maven 3+
* MySQL

## 2.Running Instructions

1. Clone project **git clone https://github.com/felipe-dibernardi/javaee-payara-micro.git**

2. Run **mvn clean install** at pom.xml level folder

3. Create project and test database structure using **init.sql** script located at {{PROJECT\_PATH}}/src/main/sql/init.sql

4. To run Unit tests, use UnitTest profile: **mvn -PUnitTest clean package**

5. Create an UberJar using **java -jar {{MVN\_.M2\_PATH}}/repository/fish/payara/extras/payara-micro/4.1.2.181/payara-micro-4.1.2.181.jar --deploy {{PROJECT\_PATH}}/target/condominio-1.0.0-SNAPSHOT.war --domainConfig {{PROJECT\_PATH}}/src/main/config/domain.xml --outputUberJar {{APPLICATION\_NAME}}.jar**

6. Run **java -jar {{APPLICATION\_NAME}}.jar**

7. EndPoints will be deployed at context root **localhost:8080/condominio-1.0.0-SNAPSHOT/webresources/**

## 3. API Documentation

### 3.1. Residents

#### 3.1.1. List All Residents

Get a list of all Residents.

* **URL**

  /residents/

* **Method:**

  GET

* **Success Response:**

  * **Code:** 200 OK
  * **Content:** [{apartment : [integer], block : [alphanumeric], document : [alphanumeric], email : [alphanumeric], id : [integer], name : [string] }]

* **Error Response:**

   * **Code:** 401 UNAUTHORIZED

#### 3.1.2. Find Resident by Id

Find a Resident by Id.

* **URL**

  /residents/{id}

* **Method:**

  GET

*  **URL Params**

   **Required:**

   `id=[integer]`

* **Success Response:**

  * **Code:** 200 OK
  * **Content:** {apartment : [integer], block : [alphanumeric], document : [alphanumeric], email : [alphanumeric], id : [integer], name : [string] }

* **Error Response:**

   * **Code:** 401 UNAUTHORIZED

#### 3.1.3. List Residents By Params

List Residents by specific Params.

* **URL**

  /residents/filter?name={name}&email={email}&block={block}&apartment={apartment}&document={document}&phone={phone}

* **Method:**

  GET

*  **URL Params**

   **Optional:**

   `name=[string]`
   `email=[alphanumeric]`
   `block=[alphanumeric]`
   `apartment=[integer]`
   `document=[alphanumeric]`
   `phone=[alphanumeric]`


* **Success Response:**

  * **Code:** 200 OK
  * **Content:** [{apartment : [integer], block : [alphanumeric], document : [alphanumeric], email : [alphanumeric], id : [integer], name : [string] }]

* **Error Response:**

   * **Code:** 401 UNAUTHORIZED

#### 3.1.4. Post Resident

      * **Code:** 401 UNAUTHORIZED
